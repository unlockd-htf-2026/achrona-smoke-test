# The Curse of Achrona — your team's runner (Hack the Future 2026)

Welcome! This is your starting repo. Some gameplay features are **stubbed out**
and their tests are **failing (RED)** on purpose — turning each one **GREEN** is a
challenge, and each green lights up your node in the shared world.

## Get going
1. Click **Use this template** → create your team's repo.
2. `flutter pub get`
3. See **[CHALLENGES.md](CHALLENGES.md)** for the task list.
4. Run one challenge locally: `./challenge c1-stepy`
5. Push — GitHub Actions verifies your green against the org and records your proof.

The game runs from the first second (a stubbed feature is just visibly inert);
you implement the stubbed functions to bring it fully alive.
